# Layout de Página com Flexbox

Este é um projeto simples de layout de página web utilizando Flexbox para criar um design responsivo. O tema é inspirado no Corinthians, com um cabeçalho, menu de navegação, conteúdo principal, barra lateral e seção de cards.

## Descrição

O projeto demonstra o uso de Flexbox para organizar elementos em um layout de página. Inclui um cabeçalho com logo do Corinthians, navegação, seções de conteúdo e um rodapé. É um exemplo básico para aprendizado de CSS Flexbox.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página
- **CSS3**: Estilização e layout com Flexbox
- **Google Fonts**: Fontes Roboto e Ubuntu

## Como Executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em qualquer navegador web moderno.
3. O layout será exibido diretamente no navegador.

## Funcionalidades

- Layout responsivo com Flexbox
